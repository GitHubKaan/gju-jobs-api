# GJU Job API
Made by Kaan, Dilan and Khanh

## Setup:
1. Create .env, .env.production, .env.testing file, use .env.example as a template and configure
2. Configure PostgreSQL
3. Configure Redis
4. npm i
5. npm run test
6. npm run dev

## Important:
Always run **npm run test** before pushing to the master branch.
